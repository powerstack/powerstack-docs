          Cookie Get or set a cookie

          cookie(string $name, string $value, int $expires, string $path, string $domain, bool $secure, bool $httponly) : mixed
          see \Powerstack\Core\Cookie

          Parameters

          $name

          string

          Name of cookie to get or set.

          $value

          string

          Value to set cookie to. (optional, only use if setting a cookie)

          $expires

          int

          Length of time a cookie is valid for in seconds. (optional, default is 3600)

          $path

          string

          The path on the server in which the cookie will be available on. (optional, default is /)

          $domain

          string

          The domain that the cookie is available to. (optional, default is '')

          $secure

          bool

          Indicates that the cookie should only be transmitted over a secure HTTPS connection from the client. (optional, default is false)

          $httponly

          bool

          When TRUE the cookie will be made accessible only through the HTTP protocol.(optional, default is false)

          Returns

          mixedcookie value if getting cookie or true/false when setting a cookie
